# Australian-Real-Estate-Portfolio-Optimization
This project tackles underperformance in Australian commercial real estate, reallocating $10.57M into high-growth markets like Brisbane , boosted IRR from 2.62% to 12.19%, almost 5x. Sensitivity analysis shows returns remain above 10%, ensuring resilience in a dynamic market.
Calculated Columns:
Return on Investment (ROI)
IRR (Internal Rate of Return)
Vacancy Percentage
Debt Ratio
Debt Service Coverage Ratio (DSCR)
Net Operating Income (NOI)
EBITDA
Gross Yield
Cap Rate
Free Cash Flow
Risk Score (based on weighted vacancy, debt, and interest rate factors)

Key Pivot Table Insights:
Average ROI by city
Average Vacancy Percentage by city
Average DSCR by city
Average Cap Rate by city
Risk score identification of top underperformers
